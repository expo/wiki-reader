# Wiki Reader (Expo)

A clone of [WikiReader](https://github.com/nsh07/WikiReader) by [@nsh07](https://github.com/nsh07), rewritten from Jetpack Compose to Expo UI using `@expo/ui/jetpack-compose`.

All credit for the original app design and functionality goes to [nsh07](https://github.com/nsh07) and the [WikiReader](https://github.com/nsh07/WikiReader) project.
